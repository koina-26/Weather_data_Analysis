 🌦️ Weather Data Analysis Project

 📌 Project Overview

This project focuses on analyzing weather data using **Python** and **Pandas**.
The main goal is to explore the dataset, clean it, and extract meaningful insights about weather patterns like temperature, humidity, and wind speed.

# 🛠️ Tech Stack

* 🐍 **Python** → Programming language
* 📊 **Pandas** → Data analysis library
* ☁️ **Google Colab** → Online development environment

 # 📂 Dataset

* CSV file containing weather information
* Includes features like:

  * 🌡️ Temperature
  * 💧 Humidity
  * 💨 Wind Speed
  * 👀 Visibility
  * 🌥️ Weather Condition

# 🔑 Key Concepts (Easy Explanation)

### 📊 Data Basics

* `read_csv()` → Load dataset
* `head()` → View first rows
* `shape` → Rows & columns
* `columns` → Column names
* `dtypes` → Data types

### 🔍 Data Exploration

* `unique()` → Unique values
* `nunique()` → Count unique values
* `value_counts()` → Frequency of values
* `info()` → Dataset summary
* 
### 🧹 Data Cleaning

* `isnull()` → Check missing values
* `notnull()` → Non-missing values
* `rename()` → Rename column

### 🔎 Data Filtering

* Filter rows based on conditions
* Example: Weather = "Clear"
* Use AND (`&`) / OR (`|`) conditions

### 📊 Data Analysis

* `groupby()` → Group similar data
* `mean()` → Average
* `std()` → Standard deviation
* `var()` → Variance
* `agg()` → Multiple operations (min, max)
